102  def parse_insert(self, keys, values, __table_name__, insert_str, values_str, symbol='%s',
103  sql_format='%s %s (%s)%s(%s)'):
104  """
105  实现此方法可自定义sql生成模式
106 
107  keys:包含了所有需要解析的字段名
108  values:包含了所有需要用到的字段的值

110  insert_str:insert的字符串
