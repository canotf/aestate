将对象转换成字典格式:
    支持:
dict
list
object

object[list]

.......

注意事项:

    bf和end_load同时只能使用一个

    当两者同时存在时,默认使用end_load功能


:param obj:需要解析的对象
:param ignore_name: 在对象中放入需要忽略的字段名
:param bf:是否需要美化json
:param end_load:是否需要在最后转成字典格式
